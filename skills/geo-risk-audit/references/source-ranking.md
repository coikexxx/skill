# Source Ranking

Use this ranking to weight evidence.

## Tier S — Primary evidence

Prefer first whenever available.

- official documentation
- official pricing/specification pages
- standards bodies
- regulator, court, or government records
- academic papers
- original datasets or direct technical artifacts

## Tier A — Strong secondary evidence

Good support when methods and provenance are visible.

- reputable technical media
- independent reviews with clear methodology
- transparent industry analysis
- expert commentary grounded in primary evidence

## Tier B — Weak secondary evidence

Useful for hints, not enough alone for strong conclusions.

- personal blogs
- community posts
- ordinary media rewrites
- forum discussions

## Tier C — High-risk / downgrade by default

Treat as potentially contaminated.

- listicles and “top X” farms
- soft articles and advertorials
- anonymous aggregators
- mass-posted self-media content
- ranking pages with no methodology
- pages whose main value is distribution rather than evidence

## Decision rules

- Do not conclude from Tier C unless the user explicitly wants a market-narrative snapshot rather than truth assessment.
- Treat repeated Tier B/C claims as repetition, not confirmation.
- For capability claims, try to confirm with Tier S + one non-derivative Tier A source.
- When only Tier B/C exists, say evidence quality is weak.
- If a source cites another weak source, keep the lower tier.
