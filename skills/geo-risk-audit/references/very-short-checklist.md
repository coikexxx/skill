# Very Short Checklist

Use this when the task is high-risk, time-limited, or likely to drift into a soft, vague answer.

## 5-line output

- **Risk**: low / medium / high
- **Trust**: strongest source types used
- **Suspect**: main contamination signal
- **Safe takeaway**: what can actually be said now
- **Next check**: what to verify before stronger conclusions

## Rules

- Keep each line short.
- Do not name a single winner unless evidence is clearly strong.
- If sources are mostly weak, say so directly.
- If evidence is mixed, give a shortlist or conditional conclusion.
