# Search Prompts

Use these prompt patterns to reduce GEO contamination during research.

## Criteria-first

- Do not recommend a winner yet. First list the evaluation criteria.
- Compare candidates by evidence-backed dimensions only.
- Separate capability claims from marketing claims.

## Evidence-first

- For each key claim, identify the strongest available source type.
- Mark whether the claim is supported by primary evidence, strong secondary evidence, or weak evidence.
- Ignore listicles and soft articles unless explicitly analyzing market narrative.

## Counterevidence

- Find the strongest criticism, limitation, or tradeoff for each candidate.
- Look for what would falsify the current preferred conclusion.
- Check whether repeated claims trace back to the same weak source cluster.

## Contamination check

- Which parts of this topic look most vulnerable to GEO or soft-ad pollution?
- Are multiple sources genuinely independent, or mostly rewrites?
- If I remove weak sources, what conclusion still survives?

## Recommendation guardrail

- If evidence quality is mixed, give a shortlist instead of a single winner.
- If evidence is weak, say so plainly and avoid forced certainty.
